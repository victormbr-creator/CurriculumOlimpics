# Curriculum Olimpics

Juego web en pixel art para desbloquear el CV interactivo de Víctor Benítez.

## Estructura

```text
index.html
css/style.css
js/game.js
assets/
```

## Cómo correrlo

Abre `index.html` en tu navegador.

## Controles

- Flechas: mover al atleta por el estadio
- E: participar / entrar al podio
- C: correr o acelerar en pruebas
- Space: saltar o activar timing
- D: arquear el cuerpo en salto de altura
- ESC: regresar al estadio

## Pruebas

1. Salto de Longitud
2. Lanzamiento de Martillo
3. Salto con Vallas
4. Salto de Altura

Al completar cada prueba aparece una celebración con serpentinas y el podio actualiza cuántas pruebas faltan para desbloquear el CV.
